## 6. สถานะคิว (Queue State Machine)

> **[TODO: ยืนยันว่าสถานะเหล่านี้ครบถ้วนตามธุรกิจจริง]**

```
                    ┌─────────────────────────────────────────┐
                    │                                         │
                    ▼                                         │
  [ออกคิว] ──► pending ──► processing ──► completed      (โอนคิว)
                 │              │                             │
                 │              ├──► transferred ─────────────┘
                 │              │     (สร้างคิวใหม่)
                 │              │
                 ▼              ▼
              on_hold      cancelled
                 │
                 ▼
              skipped
```

**คำอธิบายสถานะ:**

| สถานะ | ความหมาย | เปลี่ยนไปได้ |
|---|---|---|
| `pending` | รอเรียก | processing, on_hold, cancelled |
| `processing` | กำลังให้บริการ | completed, transferred, cancelled |
| `completed` | เสร็จสิ้น | — (Final State) |
| `on_hold` | พักคิว (ลูกค้าไม่มา) | pending (เรียกซ้ำ), skipped |
| `skipped` | ข้ามแล้ว | — (Final State) หรือ **[TODO]** กลับมาเป็น pending ได้? |
| `transferred` | โอนไปบริการอื่น | — (Final State, มีคิวใหม่แทน) |
| `cancelled` | ยกเลิก | — (Final State) |
